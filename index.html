📖 Как настроить GitHub

  1. Создайте репозиторий на GitHub (например, "samples")
  2. Создайте Personal Access Token:
    GitHub Settings → Developer settings → Personal access tokens → Tokens (classic)
    Выберите права: repo (полный доступ к репозиториям)
  3. Вставьте токен, владельца, название репозитория и ветку
  4. Нажмите "Подключиться к GitHub"
  5. Создавайте папки и загружайте файлы - они появятся в вашем репозитории!

Файлы будут доступны по ссылкам вида:
https://raw.githubusercontent.com/username/repo/branch/folder/file.wav
